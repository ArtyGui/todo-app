# To-Do List Application

Aplicação web completa para gerenciamento de tarefas, desenvolvida com Node.js e React.

## Funcionalidades

- **CRUD Completo**: Criar, listar, editar e excluir tarefas
- **Campos Personalizados**: Título (obrigatório), descrição e data de vencimento
- **Controle de Status**: Marcar tarefas como pendentes ou concluídas
- **Filtros**: Visualizar todas, pendentes ou concluídas
- **Estatísticas**: Contadores de tarefas por status
- **Design Responsivo**: Interface adaptável para diferentes dispositivos
- **Tempo Real**: Atualizações instantâneas na interface
- **UI/UX Moderna**: Design intuitivo e atraente

## Tecnologias Utilizadas

### Backend
- **Node.js** - Runtime JavaScript
- **Express.js** - Framework web
- **SQLite** - Banco de dados
- **CORS** - Controle de acesso entre origens

### Frontend
- **React** - Biblioteca para interface
- **Tailwind CSS** - Framework CSS utilitário
- **Lucide React** - Ícones modernos
- **JavaScript ES6+** - Linguagem de programação
